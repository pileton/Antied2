Cute Moodle v1.4.4

✏️

Description: Keeps a temporary record of deleted messages and any edits until you reload the app. Deleted messages turn ephemeral red with a "This message is deleted!" notice, preserve attachments, and include a dismiss button. Edits are logged in-memory.

Installation

  1. Clone or download this repo to your computer.
  2. Enable GitHub Pages in repo settings (source: Deploy from a branch, e.g., main).
  3. In Vendetta (Discord mod), go to Plugins > Add Plugin.
  4. Paste the direct URL: https://YOUR_USERNAME.github.io/REPO_NAME/index.jsx (replace with your details).
  5. Enable the plugin and test by deleting/editing messages!

Note: Vendetta is for older Discord versions; test compatibility. Reload app to clear records.

Files

  • manifest.json: Plugin metadata.
  • index.jsx: Main plugin code (JSX/JS patches for MESSAGE_DELETE/UPDATE).
